# 📌 QR Code Generator (Python + Tkinter)

A simple desktop application to generate QR codes from text or URLs using Python. Built with **Tkinter** for the GUI and **qrcode** for QR generation.

---

## 🚀 Features
- Generate QR codes instantly  
- Supports text and URLs  
- Save QR codes as `.png` files  
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack
- Python  
- Tkinter (GUI)  
- qrcode library  
- Pillow (PIL)
